# Voeux_ACS
Projet ACS

Réaliser une carte animée en ligne pour les voeux de la nouvelle année de l'ACS, à partir des éléments graphiques de l'ACS (charte graphique, logo d l'ACS en vectoriel)

- un travail graphique cohérent doit-être fait (entre l'animation et l'habillage)
- l'animation doit-être en SVG
- elle doit contenir au moins 2 animations
- l'animation doit être intégrée dans une page HTML responsive
